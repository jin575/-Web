# .githooks（再発防止フック）

親リポジトリ（自社サイト = 医療のみちしるべ）に、テンプレなどの
**独立したGitHubリポジトリのフォルダ**を誤ってコミットする事故を防ぐための
Gitフックを置いています。

## 有効化（クローンごとに一度だけ）

```bash
git config core.hooksPath .githooks
```

これで `.githooks/pre-commit` が有効になります。

## 動作

`git commit` の直前に、ステージへ「埋め込みGitリポジトリ（gitlink）」が
混じっていないか自動チェックします。混じっていればコミットを中止します。

## 運用ルール

- 新しいテンプレサイトは必ず **`テンプレサイト/` の中**に作成する。
  - `テンプレサイト/` は `.gitignore` 済みのため、中に独立リポジトリを
    置いても親リポジトリには一切影響しません。
- 万一ルート直下などに作ってしまっても、上記フックが誤コミットを止めます。
